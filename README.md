# tf-output-hello
a repo that only does hello as output

# How to use this repo

Clone this repo
```
git clone https://github.com/kikitux/tf-output-hello.git
```

Change directory
```
cd tf-output-hello
```

Terraform init
```
terraform init
```

Terraform apply
```
terraform apply
```
